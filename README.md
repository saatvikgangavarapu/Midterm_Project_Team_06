## Project Overview:
This midterm university model project is a Java application built using Netbeans and Swing GUI. The objective is to build a well-connected application that simulates a university application, serving mutliple entities that exist in a typical university. Our project implements the administrators, the faculty, and the students. By implementing a role-based access, we were able to separate the functionalities all through the same application. This allows the different users to interact with the gui elements that apply only to them, rather than having to navigate through panels that they don't use in their day-to-day operations.
